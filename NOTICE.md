## Data sources

The tool fetches player data at runtime from third-party fan APIs, which are
not affiliated with this project and have their own terms:

- Wise Old Man — https://wiseoldman.net/
- TempleOSRS — https://templeosrs.com/
- OSRS Wiki (icon images, loaded by URL) — https://oldschool.runescape.wiki/

No player data is collected or stored by this project; it is kept locally in
the user's own browser.
